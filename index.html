CSS

Que es:

CSS (Cascading Style Sheets, por sus siglas en inglés) es un lenguaje de hojas de estilo utilizado para describir la presentación (diseño y formato) de un documento HTML o XML. En otras palabras, CSS permite separar la estructura del contenido (HTML) de su apariencia visual (colores, fuentes, márgenes, espaciados, disposición, etc.).

    ¿Qué hace CSS?

  • Establece el diseño visual: Define cómo deben verse los elementos de una página web, incluyendo colores, fuentes, márgenes, bordes, tamaños y la disposición de los elementos en la página.
  • Controla la disposición de los elementos: Permite organizar y alinear elementos como texto, imágenes, formularios, botones, etc., en una página web. Se pueden crear diseños responsivos que se adapten a diferentes tamaños de pantalla.
  • Mejora la experiencia del usuario (UX): Al tener un control total sobre la apariencia, se pueden crear interfaces de usuario atractivas y fáciles de usar, lo que hace que la experiencia de navegación sea más agradable

      • Importancia de CSS en el desarrollo web:

  • Separa contenido y presentación: CSS permite mantener el contenido (HTML) y el diseño (CSS) separados. Esto facilita la creación y mantenimiento de sitios web, ya que los cambios en el diseño no afectan al contenido y viceversa.
  • Consistencia en el diseño: Con CSS, se pueden aplicar los mismos estilos a varias páginas de un sitio web, garantizando un diseño consistente a lo largo de toda la página web sin necesidad de repetir el mismo código.
  • Responsividad: Con CSS, se pueden crear diseños que se adapten a diferentes tamaños de pantalla, desde dispositivos móviles hasta pantallas de escritorio, mejorando la accesibilidad del sitio web en diferentes dispositivos.
  • Mejora del rendimiento: CSS puede optimizar el rendimiento de una página web. Al separar la presentación del contenido, los navegadores pueden almacenar en caché los archivos CSS, lo que hace que el sitio cargue más rápido.
  • Mejor accesibilidad y usabilidad: Gracias a CSS, los diseñadores pueden mejorar la accesibilidad y la usabilidad de un sitio web. Por ejemplo, se puede ajustar el contraste de colores, el tamaño de la fuente o crear efectos visuales que faciliten la navegación.
  • Estilo y personalización: CSS ofrece una gran variedad de opciones para personalizar la apariencia de un sitio web, desde la elección de fuentes hasta la creación de animaciones y transiciones. Esto permite a los desarrolladores y diseñadores crear experiencias únicas para los usuarios.

    • JavaScript

    Explicacion breve de JavaScript:

    JavaScript es un lenguaje de programación esencial en el desarrollo web moderno. Su principal función es añadir interactividad y dinamismo a las páginas web, lo que mejora significativamente la experiencia del usuario (UX). A diferencia de HTML y CSS, que se encargan de la estructura y el diseño de la página, JavaScript permite manipular el comportamiento de los elementos en tiempo real.

      Importacia de JavaScript en la web:

  • Interactividad: JavaScript es el motor de la interactividad en las páginas web. Permite a los desarrolladores agregar funcionalidades dinámicas como formularios interactivos, botones que cambian de color al pasar el ratón sobre ellos, menús desplegables, validación de formularios en tiempo real, entre otros. Sin JavaScript, las páginas web serían estáticas y no ofrecerían una experiencia atractiva.
  • Manipulación del DOM (Document Object Model): JavaScript permite modificar el DOM, que es la estructura interna de una página web. Con esto, se pueden cambiar elementos de la página sin necesidad de recargarla, lo que se traduce en una experiencia más rápida y fluida. Puedes agregar, eliminar o modificar elementos HTML, lo que permite crear aplicaciones web dinámicas.
  • Desarrollo de aplicaciones web interactivas (SPA): JavaScript es fundamental para construir Single Page Applications (SPA), que son aplicaciones web que cargan una sola página HTML y luego actualizan dinámicamente su contenido sin recargar toda la página. Esto permite una experiencia de usuario más rápida y fluida, típica de aplicaciones como Gmail, Facebook, Twitter, etc.
  • Manipulación de eventos: JavaScript permite detectar y reaccionar a eventos, como clics, desplazamiento (scroll), teclas presionadas y cambios de valor en un formulario. Esto facilita crear aplicaciones web que respondan en tiempo real a las acciones del usuario.
  • Validación de formularios: JavaScript permite validar los datos de los formularios antes de enviarlos al servidor, lo que ayuda a evitar errores comunes y mejora la seguridad. Esto permite comprobar que los datos son correctos (por ejemplo, que el correo electrónico tiene un formato válido) sin tener que hacer una nueva solicitud al servidor.
  • Integración con APIs: JavaScript permite conectar y consumir APIs (Interfaz de Programación de Aplicaciones) para interactuar con otros servicios y obtener datos dinámicamente, como mostrar el clima actual, las últimas noticias o los resultados deportivos en una página web. Las llamadas a APIs se pueden hacer utilizando herramientas como Fetch o AJAX, lo que permite obtener información del servidor sin recargar la página.
  • Desarrollo de juegos y aplicaciones multimedia: JavaScript también se usa para desarrollar juegos web y aplicaciones multimedia, como animaciones y gráficos interactivos. Con bibliotecas como Three.js para gráficos 3D o PixiJS para juegos 2D, JavaScript permite crear experiencias visuales ricas directamente en el navegador.
  • Mejora del rendimiento con tecnologías modernas: JavaScript es crucial para aprovechar tecnologías web avanzadas como WebSockets (para comunicación en tiempo real), Web Workers (para ejecutar tareas en segundo plano sin bloquear la interfaz de usuario) o Service Workers (para crear aplicaciones web progresivas y almacenar en caché recursos para uso sin conexión).
    • Frameworks y bibliotecas populares: Existen muchas herramientas basadas en JavaScript que facilitan el desarrollo de aplicaciones web modernas. Algunas de las más populares incluyen:
    • React.js: para construir interfaces de usuario interactivas.
    • Vue.js: un framework progresivo para interfaces de usuario.
    • Angular: un framework para desarrollar aplicaciones web de una sola página.
    • Node.js: permite usar JavaScript en el lado del servidor, lo que facilita el desarrollo full-stack con un solo lenguaje de programación.

    border-radius Se usa para redondear las esquinas de un elemento. Puedes aplicarla a cualquier elemento con bordes, como divs, imágenes, botones, etc. border-radius: 10px,border-radius: 50%, border-radius: 10px 20px 30px 40px,border-radius: 10px 20px. .{border-radius: 20px;} MDN
    display controla cómo se muestra un elemento HTML en la página. Define si un elemento aparece como bloque, en línea, oculto, u otras formas más avanzadas como flexbox o grid. block, inline,inline-block, none,flex,grid,inline-flex,inline-grid. .{display: inline;} MDN
    flexbox (display: flex;) Es un modelo de diseño en CSS que se usa para organizar, alinear y distribuir elementos dentro de un contenedor de forma flexible, incluso si su tamaño es dinámico o desconocido. flex, inline-flex,row, row-reverse, column, column-reverse,flex-start, flex-end, center, space-between, space-around,10px, 1rem.. .{display:felx; justify-content:space-between;} MDN
    position Controla cómo se posiciona un elemento en la página web. Define si un elemento se coloca en el flujo normal del documento o si puede estar fijo, flotando o superpuesto. static (por defecto),relative,absolute,fixed,sticky. .{position: static;} DEV
    box-shadow Se denominan sombras sobre cajas a las sombras en CSS que se pueden crear en una etiqueta o elemento HTML. Para ello, se utiliza la propiedad box-shadow, que funciona de forma muy similar a la que vimos en las sombras de texto. 10px (derecha),-5px (arriba),15px (suave),5px (agrandar),rgba(0, 0, 0, 0.5) (negro semitransparente). .{box-shadow: 10px 10px 15px rgba(0, 0, 0, 0.5);} MDN
    overflow Controla cómo se maneja el contenido desbordado (que excede el tamaño del contenedor). Es útil cuando el contenido es más grande que el área visible de un div, sección, u otro elemento HTML. visible,hidden,scroll,auto,clip. .{overflow: visible;} MDN
    z-index Es una propiedad de CSS que se usa para establecer la visibilidad y el posicionamiento del contenido superpuesto. Cuando dos elementos se superponen, el z-index determina qué elemento aparece encima del otro. Valor positivo (+),Valor negativo (-),Por defecto (z-index: auto;). .{z-index:2;} MDN
    opacity Controla el nivel de transparencia de un elemento. Permite hacer que un elemento sea más o menos visible, variando entre completamente opaco y totalmente transparente. 1:Completamente visible (100% opaco),0: Totalmente transparente (invisible),0.5:50% de transparencia (semi-transparente). .{opacity: 0.5;} MDN
    margin y padding En CSS, tanto el padding como el margin añaden espacio a los elementos de la web, pero cada uno aporta resultados diferentes. El padding es el espacio entre el contenido de un elemento y el borde. Por otra parte, el margin constituye el espacio más externo del elemento. .10px,20px, 30px, 40px. .{margin:10px 20px padding: 15px;} MDN
    color Define el color del texto de un elemento. red, blue, #ff6347, rgb(255, 99, 71), etc. {color: blue;} MDN
    background-color Define el color de fondo de un elemento. red, #fff, rgba(0, 0, 0, 0.5), etc. {background-color: #f0f0f0;} MDN
    width Establece el ancho de un elemento. 100px, 50%, auto {: 200px;} MDN Web Docs
    line-height Define la altura de las líneas de texto. 1.5, 20px, normal {: 1.5;} MDN Web Docs
